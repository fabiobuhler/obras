# Gestão Obras

Sistema de controle de obras, financeiro, apontamentos, equipamentos, funcionários e permissões por módulo.

## Deploy

Este projeto é publicado via GitHub Pages.

URL esperada:

https://fabiobuhler.github.io/obras/

## Ambiente

Configurar no GitHub Actions:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

Não incluir chaves sensíveis no repositório.
